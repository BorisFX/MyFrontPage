Персональный сайт-портфолио, демонстрирующий мои навыки как Senior .NET Developer.

## Технологии
- HTML5
- CSS3
- JavaScript

## Структура проекта
- `index.html` - главная страница
- `resume.html` - резюме
- `contact.html` - контактная информация
- `login.html` - страница входа
- `css/` - стили
- `js/` - скрипты JavaScript
